# Futu OpenAPI — Python SDK Samples

Python examples for the [Futu OpenAPI SDK](https://openapi.futunn.com/futu-api-doc/).

## Quick Start

```bash
pip install futu-api
python examples/00_connect_ha/main.py
```

## Examples

42 examples covering quote and trade APIs. See [examples/README.md](examples/README.md) for the full index.

All examples use `examples/connect.py` for HA gateway selection and RSA encryption configuration — no hardcoded IPs.
